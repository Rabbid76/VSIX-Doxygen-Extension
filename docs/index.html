Document the source code

Crate local doxygen documantation

A local doxygen html documentation for the .dox files, but not for the source code itself, will be generated when building the project .\p\1400\dox_src\DoxPreview.vcxproj. The generated HTML documentation can be found at .\p\1400\dox_src\doxygen_preview\html\index.html.

Add an image to source code documentation

Images to be embedded in the doxygen documentation must be contained in the directory ".\p\1400\dox_src\images" or a subdirectory thereof. The supported image types are .bmp, .jpg, .png, .gif, .tif and .svg.
To embedded the image use the special doxygen command @image.

e.g.

/ ****************************************************************** // **
\brief Foo
\image html my_image.svg "my image caption"
\author anyone
\date 01.01.2017
\version 1.0
/
void Foo( void )
{
...
};

If Doxygen Preview Extension is installed the image will be displayed in the quicktip of the special doxygen command @image html my_image.svg.
In order for the images to be displayed, the directory ".\p\1400\dox_src\images" is analyzed when Visual studio is started. All image file names are managed in a dictionary cache. If a new image is added, the cache has to be updated: Extras -> Options -> Doxygen Extensions -> Doxygen configuration -> Update Cache.

Create Dot Graph

cf Gaphviz - Graph Visualization Software [http://www.graphviz.org/]
cf Doteditor - [http://vincenthee.github.io/DotEditor/]

\dot
digraph "UML diagram"
{
// INTERACTIVE_SVG=YES
edge [fontsize="10", style="solid", fontname="Helvetica"]
node [shape=record, fontname="Helvetica", fontsize="10"];
// inheritance
edge [dir="back", style="solid", arrowhead="none", arrowtail="onormal", color="midnightblue"]
A -> B1
A -> B2
A -> B3
// association
edge [dir="front", style="dashed", arrowhead="vee", arrowtail="none", color="grey25"]
B1 -> C1 [label="assoziation", headlabel="*", taillabel="1"]
// aggregation
edge [dir="back", style="solid", arrowhead="none", arrowtail="odiamond", color="grey25"]
B2 -> C2 [label="aggregation", headlabel="*", taillabel="1"]
// composition
edge [dir="back", style="solid", arrowhead="none", arrowtail="diamond", color="grey25"]
B3 -> C3 [label="composition", headlabel="*", taillabel="1"]
}
\enddot
