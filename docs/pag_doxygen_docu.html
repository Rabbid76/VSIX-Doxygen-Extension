Installation

To install Doxygen download Doxygen Windows binaries and follow the Doxygen installation guide. To create related documents you can use any text editor. How to setup Visual Studio Code, or Visual Studio for Doxgen support is described later in this document.

Doxygen Requirements

  • Doxygen [http://www.stack.nl/~dimitri/doxygen/download.html]
    \\hydra\DevTools\msw\Doxygen
  • Graphviz [http://www.graphviz.org/Download.php]
    \\hydra\DevTools\msw\Graphviz
  • Mscgen [http://www.mcternan.me.uk/mscgen/]
    \\hydra\DevTools\msw\Mscgen
  • PlantUML [http://plantuml.com/download]
    \\hydra\DevTools\msw\PlantUML

Doxygen Manual

In the Doxygen Documentation can be found documantation about documenting the code, adding documents in Markdown format, including formulars, tables, graphs and diagrams and further.

Special Commands

The following is an incomplete list of interesting commands. See the full documentation at Special Commands.

General Special Commands

  • \ref ["(text)"]
  • \sa { references }
  • \code [ { } ] ... \endcode
  • \f[ ...\f]
  • \image ["caption"] [=]
  • \dot ["caption"] [=] \enddot

Special Commands for source code documentation

  • \brief { brief description }
  • \details { detailed description }
  • \return { description of the return value }
  • \bug { bug description }
  • \todo { paragraph describing what is to be done }

Special Commands for related page documentation

  • \mainpage [(title)]
  • \page (title)
  • \tableofcontents
  • \section <section-name> (section title)
  • \subsection <subsection-name> (subsection title)
  • \subsubsection <subsubsection-name> (subsubsection title)
  • \paragraph <paragraph-name> (paragraph title)
  • \attention { attention text }
  • <em> italics </em>
  • <b> bold </b>
  • <tt> typewriter </tt>
  • \copydoc <link-object>
  • \copybrief <link-object>
  • \copydetails <link-object>

Special Commands for referencing code in related page documentation

  • \headerfile <header-file> [ <header-name> ]
  • \class [ <header-file> ] [ <header-name> ]
  • \defgroup (group title)
  • \ingroup ( [ ] )

Doxygen Configuration

A configuration file is a free-form ASCII text file with a structure that is similar to that of a Makefile, with the default name Doxyfile. It is parsed by Doxygen.

Recommended Doxygen Configuration:

EXTENSION_MAPPING = he=C \
hi=C \
glsl=C
MARKDOWN_SUPPORT = YES
INPUT_ENCODING = ISO-8859-1
FILE_PATTERNS = *.dox
RECURSIVE = YES
EXCLUDE_PATTERNS = *\.vscode* \
\.git* \
\.svn* \
\Debug* \
\Release* \
\html* \
Ifc*
EXAMPLE_PATH = .
EXAMPLE_RECURSIVE = YES
IMAGE_PATH = . \
.\images \ USE_MATHJAX = YES
MSCGEN_PATH = \\hydra\DevTools\msw\doxygen\
DOT_PATH = \\hydra\DevTools\msw\graphviz\bin\
PLANTUML_JAR_PATH = \\hydra\DevTools\PlantUML
PLANTUML_INCLUDE_PATH = ./doc/attachments/UML

Visual Studio Code

Download and install Visual Studio Code for creating Doxygen documents. Useful extension for Visual Studio Code can be downloaded at Visual Studio Code Marketplace.

Recommended Visual Studio Code settings: settings.json

{
"files.encoding": "iso88591"
}

Doxygen language support

Doxygen language support for Visual Studio Code adds adds syntax highlighting for for .dox-files to Visual Studio Code.

SVG Viewer

SVG Viewer displays .svg files and convert .svg files to .png files.

Preview

Preview is previewer of Markdown, ReStructured Text, HTML, Jade, Mermaid files, Image's URI or CSS properties for Visual Studio Code.

PlantUML

PlantUML is a plugin that preview and export PlantUML diagrams in VSCode.

Status Bar Tasks

Status Bar Tasks extension loads Visual Studio Code tasks into executable status bar commands.

Recommended Visual Studio Code tasks: tasks.json

{
// See https://go.microsoft.com/fwlink/?LinkId=733558
// for the documentation about the tasks.json format
"version": "0.1.0",
"command": "cmd.exe",
"isShellCommand": true,
"showOutput": "silent",
"args": [
"/c"
],
"tasks": [
{
"taskName": "create preview",
"suppressTaskName": true,
"args" : ["\\\\hydra\\DevTools\\msw\\doxygen\\doxygen.exe", "Doxyfile.preview"],
"showOutput": "always",
"isBuildCommand": true,
"problemMatcher": "$msCompile"
},
{
"taskName": "show preview",
"suppressTaskName": true,
"args" : ["start", ".\\doxygen_preview\\html\\index.html"],
"showOutput": "never",
"isBuildCommand": false
}
]
}

Statusbar Commands

statusbar-commands extends the status bar with own commands.
Icons are located on [https://octicons.github.com/].

Recommended Visual Studio Code settings: settings.json

"statusbar_command.commands": [
{
"text": "$(gear)",
"tooltip": "workspace settings",
"alignment": "right",
"priority": 1,
"command": "workbench.action.openWorkspaceSettings"
},
{
"text": "$(sync)",
"tooltip": "Show GLSL Preview",
"alignment": "right",
"include": ".glsl",
"command": "extension.showGlslPreview"
},
{
"text": "$(sync)",
"tooltip": "SCG Viewer: View SVG",
"alignment": "right",
"include": ".svg",
"command": "svgviewer.open"
},
{
"text": "$(sync)",
"tooltip": "Open Preview",
"alignment": "right",
".rst .rest .hrst .mmd .mermaid",
"command": "extension.previewToSide"
},
{
"text": "$(file-media)",
"tooltip": "Export bitmap",
"alignment": "right",
"include": ".wsd|.pu|.puml|.plantuml|.iuml|.plantUML",
"command": "plantuml.exportCurrent"
}
]
